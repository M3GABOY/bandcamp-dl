# bandcamp-dl
Bandcamp Batch Downloader (Archiver).

Works directly with the artist homepage of bandcamp.

# Usage:
```
usage: bandcamp-dl.py [-o OUTPUT] url

positional arguments:
  url                   URL of Video Page

optional arguments:
  -o OUTPUT, --output OUTPUT
                        Set Output Location
```

# Requirements:
```
youtube-dl

python3

requests

json

argparse

io

bs4

re

urllib
```
